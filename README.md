## About
Dive into the Pytorch ecosystem: how to load data, set up architecture, train and predict using neural nets.

## Learning Objectives

*You will be able to:*

- Learn PyTorch basics: tensors, DataLoaders, and Transforms.
- Implement regression and classification models using PyTorch.
- Explore model optimization: loss functions, gradient descent, and backpropagation.
